# Tourplan Travel API Planner Wrapper

## Install

Via Composer

``` bash
$ composer require shineklbm/tourplan
```

## Usage

``` php
$skeleton = new shineklbm\tourplan;
echo $skeleton->echoPhrase('Hello, League!');
```
